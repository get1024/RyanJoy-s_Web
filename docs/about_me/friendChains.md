::: code-group
```html
<div class="linkcard">
  <a href="[你的网站链接]" target="_blank">
    <p class="description">[你的网站名称]
      <br/>
      <span>[你的网站简介，最好不要超过30个汉字]</span>
    </p>
    <div class="logo">
        <img alt="Logo" width="50px" height="50px" src="[你的头像链接]" />
    </div>
  </a>
</div>
```
:::
